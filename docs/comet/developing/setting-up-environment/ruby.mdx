Ruby is a dynamic, object-oriented programming language known for its simplicity and productivity. It powers frameworks like Ruby on Rails and is widely used for web development, scripting, and automation tasks. Ruby's elegant syntax and robust ecosystem make it a popular choice for developers.

## Install Ruby

Installs ruby using the apt package manager.

```bash
sudo apt install ruby-full
```

## Verify Installation

After installation, verify that Ruby is properly installed:

```
ruby --version
gem --version
```

## Hello World Example

Let's create a simple hello world program to test the installation:

**Create a new directory**
```
mkdir hello_ruby
cd hello_ruby
```

**Create a Ruby file**
```
echo 'puts "Hello, Ruby!"' > hello.rb
```

**Run the program**
```
ruby hello.rb
```
