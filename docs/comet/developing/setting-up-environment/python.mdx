## Install Python

Python should already be installed on the Comet. If it's not installed the following command will install Python 3 using the apt package manager.

```bash
sudo apt install python3
```

## Verify Installation

After installation, verify that Python is properly installed:

```
python --version
pip --version
```

## Hello World Example
Let's create a simple hello world program to test the installation:

**Create a new directory**

```
mkdir hello_python
cd hello_python
```

**Create a Python file**

```
echo 'print("Hello, Python!")' > hello.py
```
**Run the program**

```
python hello.py
```
