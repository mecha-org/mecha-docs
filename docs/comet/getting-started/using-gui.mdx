The Mecha Comet operates on **Mechanix OS**, a Debian-based Linux distribution designed for handheld devices. This OS provides access to over 60,000 packages from the Debian repositories, allowing users to install and run a wide range of applications effortlessly.

## **Getting Started with the Mechanix Shell**

The **Mechanix Shell** is the graphical user interface (GUI) of the Mecha Comet. It is built using Rust and optimized for small-screen devices, offering a responsive and efficient user experience.

### **Key Features of the Mechanix Shell:**

- **High Performance:** Rust based, ensuring fast execution with minimal CPU and memory usage.
- **Touch-Friendly Interface:** Designed for the Comet's 3.4-inch capacitive touch LCD
- **Wayland-Based Rendering:** Utilizes GPU acceleration for enhanced graphical performance.
- **Open-Source UI Framework:** Developer friendly for building custom applications.

### **Accessing the GUI**

- Upon powering on, the Mecha logo appears, followed by the home screen.
- To unlock from sleep mode, swipe right over the patterned line on the screen.

## **Navigating the GUI**

### **1. Home Screen**

- Displays **Wi-Fi, battery, Bluetooth status**, and system settings.
- Shows **device IP, uptime, CPU usage, and memory graphs** for quick monitoring.
- Provides **quick access** to the **Terminal and Chrome browser**
- Use the **programmable button** to switch between recently used apps

<img src="/img/comet/device-homescreen.jpeg" alt="Device Homescreen" width="500" style={{ filter: 'brightness(2.0)' }} />

### **2. App Drawer**

- Swipe up from the bottom of the screen to open the App Drawer.
- Browse, search, and launch applications easily.

<img src="/img/comet/device-app-drawer.jpeg" alt="Device App Drawer" width="500" style={{ filter: 'brightness(2.5)' }} />

### **3. System Settings**

- Customize display settings, network connections, and system preferences.
- Access the **Settings** app by swiping down on the home screen.

### **4. Installing Applications  (add steps to install app by cmd ?)**

- Use the terminal commands:
    
    ```
    $ sudo apt update
    $ sudo apt install <package-name>
    ```
    

## **For Developers**

Developers can leverage the open-source **Mechanix UI framework** to create or port applications with: 

- Seamless GUI integration.
- Full control over appl design and interaction.
- Access to development tools and documentation.

To get started, visit the official repository and follow the setup instructions for building applications on the Mecha Comet.
