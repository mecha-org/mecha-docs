Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It is commonly used for developing server-side applications, tools, and APIs. Node.js also includes `npm`, a package manager for JavaScript libraries.

## Install using apt

To install Node.js on systems that use `apt` (such as Ubuntu or Debian), follow these steps:

**Step 1: Add the Node.js PPA (Personal Package Archive)**

NodeSource provides up-to-date versions of Node.js. Add their repository:

```bash
# Add the NodeSource repository for Node.js (replace <version> with the desired version, e.g., 20.x)
curl -fsSL https://deb.nodesource.com/setup_<version>.x | sudo -E bash -
```
For example, to install Node.js 20.x:

```
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
```

**Step 2: Install Node.js**

```
# Install Node.js and npm
sudo apt install -y nodejs
```

## Verify Installation
After installation, verify that Node.js and npm are properly installed:

```
node --version
npm --version
```

## Hello World Example
Let's create a simple Hello World program to test the installation:

**Create a new directory for your project**

```
mkdir hello_node
cd hello_node
```
**Create a new JavaScript file**

```
echo 'console.log("Hello, Node.js!");' > hello.js
```
Run the program

```
node hello.js
```
