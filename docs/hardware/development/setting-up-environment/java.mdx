## Install Java

The following commands to install Java using the `apt` package manager:

```bash
sudo apt update
sudo apt install default-jdk
```
## Verify Installation
After installation, verify that Java is properly installed:

```
java -version
javac -version
```

## Hello World Example
Let's create a simple "Hello, Java!" program to test the installation:

**Create a new directory**

```
mkdir hello_java
cd hello_java
```
**Create a Java file**

```
echo 'public class HelloJava {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}' > HelloJava.java
```
**Compile the program**

```
javac HelloJava.java
```
**Run the program**

```
java HelloJava
```
